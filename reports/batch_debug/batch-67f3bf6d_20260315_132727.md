# Incident Report: Batch 67f3bf6d

## 1. Issue
A **500 Internal Server Error** occurred in the **`payment-service`** when processing `POST /api/payments/checkout` in staging. The root cause was traced to an **unhandled exception** in the **`/api/users/:id`** endpoint of the **Node.js user service**, where:
- The endpoint did not handle cases where `userService.getById()` throws a **"User not found"** error.
- Instead of returning a **404 Not Found**, the error propagated unhandled, causing downstream failures in the payment flow (which likely depends on user data validation).

The incident was triggered when:
1. A payment request was made to `/api/payments/checkout`.
2. The payment service attempted to validate the user via the **user-service** (`GET /api/users/:id`).
3. If the user did not exist, the **user-service** crashed with a 500 error instead of returning a 404, breaking the payment flow.

---

## 2. Solution
The fix involved **adding proper error handling** to the `GET /api/users/:id` endpoint in **`node-service/src/routes/users.js`** to:
1. Catch errors from `userService.getById()`.
2. Return a **404 Not Found** for "User not found" errors.
3. Pass other errors to the global error handler via `next(error)`.

This ensures:
- The **user-service** fails gracefully when a user is missing.
- The **payment-service** receives a **404** (instead of a 500) and can handle it appropriately (e.g., reject the payment with a clear error message).

## 5. Exact Code Modified

### Before (Vulnerable Code)
```javascript
// node-service/src/routes/users.js
router.get("/:id", authenticate, async (req, res) => {
  const user = await userService.getById(req.params.id);
  res.json({ user: user.toJSON() });
});
```

### After (Fixed Code)
```javascript
// node-service/src/routes/users.js
router.get("/:id", authenticate, async (req, res, next) => {
  try {
    const user = await userService.getById(req.params.id);
    res.json({ user: user.toJSON() });
  } catch (error) {
    if (error.message === "User not found") {
      res.status(404).json({ error: "User not found" });
    } else {
      next(error); // Pass to global error handler
    }
  }
});
```
